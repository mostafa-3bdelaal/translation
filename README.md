# 🌐 Smart Translator 

A multilingual translation app built with Python and Gradio that automatically detects the input language and translates it using 
Hugging Face NLP models.

---

## 🚀 Features

- Auto language detection
- Multi-language translation
- Real-time translation
- Copy translated text
- Simple Gradio UI

---

## 🛠️ Technologies

- Python
- Gradio
- Hugging Face API
- langdetect
- requests

---

## 🌍 Live Demo

https://huggingface.co/spaces/mostafa-3bdelaal/Translation

---

## 🔑 Setup Token

Create a file named:

```env
HF_TOKEN.env
```

Add your Hugging Face token:

```env
HF_TOKEN=your_token_here
```

Get token from:

https://huggingface.co/settings/tokens

---

## ▶️ Run App

```bash
python -m gradio app.py
```
